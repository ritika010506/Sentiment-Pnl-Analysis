# Market Sentiment vs PnL Analysis

Analyze how market sentiment (Fear vs Greed) impacts trading PnL.

## Project Structure
data/
src/
notebooks/
outputs/

## Pipeline
1. Data Loading  
2. Data Preprocessing  
3. Feature Engineering  
4. Aggregation  
5. Visualization & Insights  

## Insights
- Higher PNL during fear periods
- Fear periods increased the number of trades made per day
- Greed periods led to overtrading behaviour
- Longer trades were made during fear periods
